# Rehletna – Offers & Packages Management Module

Rehletna Offers & Packages Management is an advanced module developed for a multi-agency tourism management platform.  
The module focuses on managing tourism offers, travel packages, linked activities, services, and agency-specific back-office operations.

This project was developed using **Symfony**, **Twig**, and **SQL**, with a strong focus on clean structure, modern development practices, and a user-friendly administration experience.

## 🚀 Features

- Offers and travel packages management
- Create, update, delete, and display tourism offers
- Link offers with activities and tourism services
- Agency agent back-office
- Super admin monitoring features
- Advanced search and filtering
- AI-assisted offer form generation
- AI-powered offer search
- API integrations
- Dynamic Twig-based user interfaces
- SQL database management
- Secure and structured Symfony architecture

## 🛠️ Technologies Used

- **Symfony** – Backend framework
- **Twig** – Template engine
- **SQL** – Database management
- **PHP** – Server-side development
- **CSS** – Interface styling
- **JavaScript** – Frontend interactions

## 📌 Project Context

This module is part of **Rehletna**, a cross-platform multi-agency tourism management platform.  
The goal of the platform is to centralize tourism services and simplify the interaction between travel agencies, clients, offers, activities, reservations, and communication services.

## 🎯 Module Objective

The objective of this module is to provide agencies and administrators with a robust system to manage tourism offers and packages efficiently, while improving the user experience through smart search, AI features, and service/activity linking.

## 📂 Main Functional Areas

- Offer management
- Package management
- Activity linking
- Service linking
- AI offer generation
- AI search assistance
- Agency dashboard
- Super admin monitoring
